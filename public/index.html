    Welcome to EDocCabinet API

    Welcome to the EdocCabinet API - A Document Management System API! You can use our API to access eDocCabinet API endpoints, which can give you information on how to register, login, add a new document and manage your document in our database.

    We have language bindings in JavaScript! You can view code examples in the dark area to the right. This example API documentation page was created with Slate. Feel free to edit it and use it as a base for your own API's documentation.

    Authentication

    eDocCabinet uses (username and password) to login to allow access to the API protected routes. On registering or login, a token is generated.

    eDocCabinet expects for the token to be included in all API requests to the server in a header that looks like the following:

    x-access-token: token

    To get access to the document, you have to be registered: Register and login to get a token to be inserted in the header
